# GD Faces

## API

Request URL:
```/(diff),(special)```

Example:
```/demon-easy,featured```

### diff
- na (also can be an empty value)
- easy
- normal
- hard
- harder
- insane
- demon-easy
- demon-medium
- demon-hard
- demon-insane
- demon-extreme

### special
- featured
- epic
- null (an empty value)
