## API

Request URL
```/-type-/-diff-/-special-```

### type
- basic
- demon

### diff
- na (also can be empty value)
- easy
- normal !-for basic type only
- hard
- harder !-for basic type only
- insane
- medium !-for demon type only
- extreme !-for demon type only

### special
- featured
- epic
- null (empty value)
